<h2> About the Project</h2>

<p>
<strong>ElectroZone</strong> is a modern e-commerce platform focused on selling electronic devices such as smartphones,
laptops, cameras, and tech gadgets. The project was designed with a clean UI, smooth animations,
and a responsive layout to provide an excellent shopping experience.
</p>

---

<h2> Features</h2>

<ul>
<li> Modern and responsive UI</li>
<li> Product listing with prices</li>
<li> Wishlist functionality</li>
<li> Product rating system</li>
<li> Smooth animations</li>
<li> Clean design with Tailwind</li>
</ul>

<h2> Installation</h2>

<pre>
git clone https://github.com/seu-usuario/electrozone.git
cd electrozone
npm install
npm run dev
</pre>

Open in your browser:

<pre>
http://localhost:5173
</pre>
